# Release Guide

## Create v0.1.0 on GitHub

1. Open the GitHub repository.
2. Go to `Releases`.
3. Click `Draft a new release`.
4. Click `Choose a tag`.
5. Enter:

```text
v0.1.0
```

6. Create the new tag from `main`.
7. Use this release title:

```text
Design Knowledge Engine v0.1.0
```

8. Paste the contents of [`../RELEASE_NOTES_v0.1.0.md`](../RELEASE_NOTES_v0.1.0.md) into the release description.
9. Confirm the target branch is `main`.
10. Publish the release.

## Suggested Release Description

```markdown
# Design Knowledge Engine v0.1.0

Initial open-source preview release.

Features:

- Image analysis workflow
- Markdown knowledge generation
- Knowledge base archiving
- Cross-platform support
- Automated testing
```
